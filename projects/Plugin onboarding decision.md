# Plugin onboarding decision

Decision: the setup guide should help a new user reach one useful related note before explaining settings. The risk is that people judge the plugin from an empty test note or stop at browsing results.

Start from a real draft, meeting note, research note, or decision note. Then use Connections to recover nearby context before adding durable structure.

The first useful moment should feel small: inspect one promising result, decide whether it belongs here, and keep writing.
