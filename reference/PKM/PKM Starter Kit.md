> [!When to use]

### Starter kit
> [!summary]
> A minimal PKM setup you can start today: one inbox, a few core notes, and a review loop.

## 1) Create one inbox
- One place where all captures land (notes, links, meeting scraps).
- Rule: capture first, organize later.

## 2) Create 5 core notes (or use existing)
- [[Personal Knowledge Management Overview]]
- [[Core Components of PKM]]
- [[Capturing Information]]
- [[Processing and Organizing Notes]]
- [[PKM Weekly Review]]

## 3) Run the loop (tiny)
- Daily (5-10 min): process 1-3 inbox items
- Weekly (30-60 min): clear backlog + choose one micro-output

## 4) Output scoreboard
Your PKM works when it produces:
- one decision you reuse,
- one paragraph you write,
- one checklist you improve.

## Guardrail
If you are not shipping anything, reduce capture and increase synthesis (see [[Synthesis Workflow]]).

### Failure mode and fix
Failure mode: You collect notes but do not process.

Fix: set a daily 5-minute timer and process 1-3 items. Keep the inbox small.
