> [!summary]
> Reduce friction, increase wins, then compound.

> [!When to use]
> When starting tasks is hard, you get stuck in avoidance, or you switch tasks frequently.

### Approach
> [!summary]
> Make progress feel easy to start by limiting WIP and using tiny timeboxes.

## The 3 levers
- WIP limit: keep only 3 active items (see [[3-slot Today Board]]).
- Starter sprints: break start friction (see [[Starter Sprint]] and [[Novelty Sprint Script]]).
- Visible wins: define a "first visible step" for each task.

## A daily routine (10 minutes)
- Choose 3 tasks for the board.
- Run one 5-minute starter sprint on "Now".
- Convert the task into a next action if needed (see [[Next Action Converter]]).

## Guardrails
- Keep tasks small.
- Make success visible.
- Do not use the system to punish yourself on bad days.

## Proof metric
- Days you complete 2+ slots (trend up)

### Failure mode and fix
Failure mode: You add too much structure.

Fix: keep it to the board and one sprint. Add more only if it helps.

### Covers / Does not cover
- Covers: A practical approach to start and follow through.
- Does not cover: Medical advice.
