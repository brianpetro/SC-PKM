- Concrete example: 1) You capture: "Weekly review feels heavy; need a default query set." 2) You choose a workflow template.

### Failure modes: template bloat, inconsistent fields
- Fix: keep 1 canonical template per note_kind and create "snippets" for optional add-ons.
- Fix: lock down the frontmatter schema and review it monthly.
- Symptom: 12 slightly different templates for the same note role.
