> [!summary]
> An outline with linked evidence turns notes into draft momentum.

> [!When to use]
> When you have 5-15 related notes and want to start drafting without rereading everything.

### Template
# Outline Handoff Template

> [!summary]
> A simple structure for turning a cluster of notes into a draft-ready outline.

## Working title
- {title}

## Audience and promise
- Audience:
- Promise (1 sentence):

## Core question
- What question does this answer?

## Outline
1) {section}
	 - Claim:
	 - Evidence notes:
		 - [[Note 1]]
		 - [[Note 2]]
2) {section}
	 - Claim:
	 - Evidence notes:
		 - [[Note 3]]

## Tensions / counterpoints
- What would a smart skeptic say?
- Which notes contradict or constrain the claims?

## Next action
- Write the first 150 words for section 1.

### Failure mode and fix
Failure mode: You keep outlining and never draft.

Fix: end with a 150-word drafting next action and do it immediately.

### Covers / Does not cover
- Covers: A draft-ready outline format with linked supporting notes.
- Does not cover: Full writing style guides.
