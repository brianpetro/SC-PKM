> [!summary]
> A focus mode is a default you choose once so you do not fight distractions every day.

> [!When to use]
> When you are about to start deep work or you keep losing time to notifications and context switching.

### Checklist
> [!summary]
> A checklist to make a focus environment that reduces interruptions.

## Device
- [ ] Enable a Focus/Do Not Disturb mode for work
- [ ] Allowlist only critical people/apps (if needed)
- [ ] Disable badges for email and chat
- [ ] Put phone out of reach during focus blocks

## Desktop
- [ ] Close email and chat
- [ ] Close all tabs except what you need
- [ ] Full-screen the working app
- [ ] Prepare the file you will work in before the timer starts

## Environment
- [ ] Water nearby
- [ ] Headphones or quiet space
- [ ] Timer visible

## Start rule
- [ ] Run [[Pre-Flow Checklist]] and start the timer

### Failure mode and fix
Failure mode: You build a perfect setup but never use it.

Fix: keep only the steps you will actually repeat. A 70% setup used daily beats a 100% setup used never.

### Covers / Does not cover
- Covers: Practical focus environment setup.
- Does not cover: Choosing what to work on.

## Next action
- Turn on Focus/Do Not Disturb right now and disable one non-essential notification source.
