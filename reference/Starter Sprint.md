> [!summary]
> A short sprint lowers the bar enough that starting becomes easy.

> [!When to use]
> When you are avoiding a task or you cannot get started.

### Sprint script
> [!summary]
> A short timer sprint to create momentum when starting feels hard.

## The sprint (5 minutes)
- [ ] Pick one tiny next action (use [[Next Action Converter]] if needed).
- [ ] Set a timer for 5 minutes.
- [ ] Only goal: do the first tiny step.
- [ ] When the timer ends, decide:
	- Stop (you still made progress), or
	- Keep going for another 10-25 minutes.

## Tiny steps that work
- Open the doc and write 3 bullets.
- Rename the note to match what you would search.
- Create the empty headings and fill one example.

## Failure mode
If you keep "preparing to start", make the prep the sprint:
- "Open the file and write the first sentence."

### Concrete example
Task: "Update +reference table".

Sprint: Open +reference.md and add the first 3 new rows.

If momentum appears, continue for 25 minutes.

### Failure mode and fix
Failure mode: You pick a sprint task that is too big.

Fix: the sprint task must be doable even on a low-energy day.

### Covers / Does not cover
- Covers: Starting behavior when motivation is low.
- Does not cover: Long session planning (see [[Pomodoro Technique]] or [[Deep Work]]).
