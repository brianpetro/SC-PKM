> [!summary]
> Weekly check-ins turn goals into steering, not wishing.

> [!When to use]
> During a weekly review, when you want to move KRs with small, consistent adjustments.

### Template
```
# Weekly KR Check-in ({{date:YYYY-MM-DD}})

> [!summary]
> Update key results with numbers, delta, and the next move.

## KRs
- KR1: current -> target
	- Current:

	- Next move:

	- Current:
	- Delta this week:
	- Next move:
- KR3: current -> target
	- Current:
	- Delta this week:
	- Next move:

-
```

### Failure mode and fix
Failure mode: You write status text but no numbers.

Fix: each KR must have a current value and a delta, even if the delta is zero.

### Covers / Does not cover
- Covers: A short weekly OKR update ritual.
- Does not cover: Full planning systems.
