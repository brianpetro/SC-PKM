> [!summary]
> Minimal daily loop: capture, triage, link, and choose a next action or micro-output.

> [!When to use]
> If you capture but never convert, this daily workflow shows how to triage, link, and create one small output, even when you are busy.

### Start-of-day capture and scan
- Quick scan: what is already in the inbox?
- Capture any overnight thoughts or urgent inputs.
- Pick one focus area for today (what output matters?).

### Triage new inputs into buckets
- Task: needs action (record owner/next step).
- Reference: keep as a link or source note.
- Concept: convert into a permanent note.

### Convert 1 to 3 items into permanent notes
- Rewrite in your own words.
- Ensure the note is one idea (atomic).
- Add 1-3 meaningful links.

### Identify one micro-output from notes
- One paragraph added to a draft.
- One decision record created.
- One checklist improved.

### End-of-day cleanup and reflection
- Add one link or one tag correction.
- Write a 2-line reflection:
- What mattered today?


## Next action
Pick one note you touched this week and upgrade it using this note's main rule (1 pass only).
