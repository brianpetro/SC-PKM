> [!summary]
> Copy/paste this into a new note to run a deep work block with a clear outcome and checkpoint.

> [!When to use]
> When you want deep work to produce a visible artifact instead of just time spent.

### Template
```
---
type: deep-work-block
date: "{{date}}"
start: "{{time}}"
duration_min: 50
---

> [!summary]
> Outcome for this block: 

## Outcome (one sentence)
- 

## First move (2 minutes)
- Open:
- Write:
- Remove:

## Constraints (keep it honest)
- No email/chat
- One window
- Phone away

## Work log (optional)
- 

## End checkpoint (2 minutes)
- Shipped:
- Next step:
- Block rating (1-5):
```

### How to use it
1) Create a new note.
2) Paste the template.
3) Fill the Outcome and First move.
4) Start the timer.
5) At the end, write what you shipped and the next step.

### Failure mode and fix
Failure mode: You fill the template with multiple goals.

Fix: keep exactly one outcome for the block. If you have two, schedule two blocks.

### Covers / Does not cover
- Covers: A repeatable structure for deep work sessions.
- Does not cover: Project planning across weeks (use [[PKM Weekly Review]]).
