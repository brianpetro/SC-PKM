> [!summary]
> If you cannot start, the task is probably not a next action yet.

> [!When to use]
> When you are stuck, procrastinating, or your task list is full of vague items.

### Use this block
> [!summary]
> Convert vague tasks into a next physical action you can do in 2-5 minutes.

## The converter
1) Write the vague task:
- Vague: 

2) Ask: "What would I do if I had 2 minutes and could only move this forward a little?"
- Next physical action (verb-first): 

3) Add the setup step if needed:
- If the action requires prep, write the setup as the next action instead.

## Examples
- Vague: "Work on marketing page"
	- Next action: "Open the draft note and write 5 bullet headings"

- Vague: "Fix onboarding"
	- Next action: "List the top 3 points of confusion from user feedback"

### Failure mode and fix
Failure mode: You write a next action that is still vague ("research options").

Fix: add a deliverable constraint: "write a 5-bullet comparison" or "collect 3 options and 1 tradeoff each".

- Does not cover: Full project planning.
