> [!summary]
> Daily check-in to process inbox, add links, and note what mattered today.

> [!When to use]
> If your inbox keeps growing and you lose trust, this daily review loop keeps processing small and steady, even on packed days.

### Minimal daily review steps (5 to 15 min)
- You end the session with fewer open loops than you started with.
- You created at least one link that improves future retrieval.
- Timebox: 5-15 minutes.

### Inbox triage and next action extraction
- Is this actionable?
- If yes: what is the next action, and where does it belong?
- If no: archive, delete, or convert to a note.

### Link and tag pass for new notes
- Spending the whole review tagging. Fix: treat tagging as a light pass and focus on links and actions.
- Add 1-3 meaningful links to each promoted note.
- Prefer links that explain dependencies or contrasts.

### Quick reflection prompt and capture
- What mattered today?
- What did I learn that I can reuse?
- What is one adjustment for tomorrow?

### Daily review metrics and common skips
- Turning daily review into a full planning session.
	- Fix: keep it small; planning belongs in weekly review.
- Inbox size (should trend down).


## Next action
Pick one note you touched this week and upgrade it using this note's main rule (1 pass only).
