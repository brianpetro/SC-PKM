> [!summary]
> The Pomodoro Technique uses a timer to create urgency and reduce procrastination.

> [!When to use]
> When you need help starting, or you lose focus after 10-15 minutes.

### Basic method
1) Pick one task.
2) Set a timer for 25 minutes.
3) Work until it rings.
4) Take a 5-minute break.
5) Repeat. After 4 cycles, take a longer break.

### How to make it work for real output
- Name the output for the pomodoro ("draft 2 paragraphs").
- Close inputs that will hijack attention.
- Keep a "distraction list": write interruptions down, handle later.

### Concrete example
Pomodoro 1: outline headings.
Pomodoro 2: write first section.
Pomodoro 3: add one example + one failure mode.
Pomodoro 4: edit and ship.

### Failure mode and fix
Failure mode: You restart the timer over and over without shipping.

Fix: end each pomodoro with a visible artifact (a paragraph, a commit, a decision).

### Covers / Does not cover
- Covers: A simple timeboxing method to start and sustain focus.
- Does not cover: Long deep work blocks (see [[Deep Work]]).
