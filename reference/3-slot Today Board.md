### Board
# 3-slot Today Board

> [!summary]
> A tiny WIP limit that makes days winnable.

## The board
- Now (1)
- Next (1)
- Later (1)

## Rules
- Only 3 slots total.
- You cannot add a 4th item. If you want to add one, you must remove one.
- Tasks must be verb-first and small enough to finish today.

## How to use
- Write 3 tasks in the slots.
- Start with "Now" and finish it before switching.
- If you stall, run a [[Starter Sprint]] for 5 minutes.

## Proof metric
- Slots completed per day (aim: 2-3)

### Failure mode and fix
Failure mode: You treat the board as a long list.

Fix: keep exactly 3 items. If something new arrives, decide what gets displaced (see [[Priority Displacement Rule]]).

### Covers / Does not cover
- Covers: A lightweight daily execution board.
- Does not cover: Long-term project planning.

## Next action
- Write 3 verb-first tasks in Now/Next/Later for today.
