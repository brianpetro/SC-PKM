> [!summary]
> Run this before deep work to reduce start friction and interruptions.

> [!When to use]
> Right before a deep work block, when you feel resistance or you keep getting interrupted.

### 60-second checklist
- [ ] Outcome: write one sentence for what you will ship in this block.
- [ ] First move: open the file and write the first 1-3 bullets.
- [ ] Inputs closed: close email/chat; silence notifications.
- [ ] Distractions removed: phone out of reach; only one browser window.
- [ ] Timer set: 25/45/50/90 minutes.
- [ ] Start: do the first move immediately.

### Concrete example
Outcome: "Draft 2 paragraphs explaining what the Starter Vault is."

First move: write the two section headings and one bullet under each.

### Failure mode and fix
Failure mode: You treat this as a ceremony and it becomes another reason to delay.

Fix: cap it at 60 seconds. If you cannot finish in 60 seconds, you are planning, not starting.

### Covers / Does not cover
- Covers: Starting a focus block quickly and reliably.
- Does not cover: Choosing priorities (use [[Daily Planning & Prioritization]]).

## Next action
- Before your next focus block, run the 60-second checklist and start the timer immediately.
