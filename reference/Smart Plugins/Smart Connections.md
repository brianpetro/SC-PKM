> [!summary]
> Smart Connections shows what is semantically related to the note you are in, so you can link and move forward without searching.

> [!tip] When to use
> Use this while writing, reviewing, or when you feel stuck and want the next relevant note in 10 seconds.

## Docs

- Start with the overview at [Smart Connections](https://smartconnections.app/smart-connections/).
- Do the first-win steps in [Getting Started with Smart Connections](https://smartconnections.app/smart-connections/getting-started/) or watch the walkthrough at [Smart Connections Getting Started story](https://smartconnections.app/story/smart-connections-getting-started/).
- Learn the daily loop (scan, preview, pin/hide, drag-to-link) in [Exploring the Connections view](https://smartconnections.app/smart-connections/list-feature/).
- Use semantic search when you know the question but not the keyword: [Smart Lookup](https://smartconnections.app/smart-connections/lookup/).
- Tune noise vs signal (sources vs blocks, limits, filters, scoring, reranking) in [Smart Connections settings](https://smartconnections.app/smart-connections/settings/).
- For "connections while you write", see [Inline connections](https://smartconnections.app/smart-connections/inline/) (Pro).
- For relevance inside tables/dashboards, see [Bases integration](https://smartconnections.app/smart-connections/bases/) (Pro).
- If results are empty, stale, or slow, troubleshoot indexing and embeddings in [Smart Environment settings](https://smartconnections.app/smart-environment/settings/).
