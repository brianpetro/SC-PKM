> [!summary]
> Smart Context turns scattered notes into a clean, token-aware bundle you can paste into chat, drafts, or agents.

> [!tip] When to use
> Use this when your flow breaks because gathering context takes longer than the work itself.

## Docs

- Overview and upgrade path: [Smart Context](https://smartconnections.app/smart-context/).
- Get a first win quickly: [Getting Started with Smart Context](https://smartconnections.app/smart-context/getting-started/).
- Learn link depth and predictable exports: [Smart Context Clipboard](https://smartconnections.app/smart-context/clipboard/).
- Save and reuse named packs: [Smart Context Builder](https://smartconnections.app/smart-context/builder/).
- Copy directly from the file explorer: [File Nav Actions](https://smartconnections.app/smart-context/file-nav-actions/).
- Control formatting with templates: [Smart Context settings](https://smartconnections.app/smart-context/settings/).
- Pin a reusable context manifest inside a note (including external repos/files): [Smart Context codeblock](https://smartconnections.app/smart-context/codeblock/).
- If context is missing or huge, tune indexing and exclusions in [Smart Environment settings](https://smartconnections.app/smart-environment/settings/).
