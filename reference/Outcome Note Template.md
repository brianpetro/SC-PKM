> [!summary]
> A compact template for capturing an outcome, decisions, and the next physical action.

> [!When to use]
> Use this when a note or project feels fuzzy and you need a single-page structure to make progress obvious.

### Outcome note template
Copy, paste, and fill in.

```
# Outcome
- WDLL:

# Context
- Why now:
- Constraints:
- Stakeholders:

# Decisions
- Decision 1:
	- Options:
	- Default:
	- Rationale:

# Next actions
- [ ] One small, physical step:

# Validation
- Proof of done:
```

### Tips
- Keep WDLL observable (someone else could verify it).
- Decisions become input for better next actions.

## Next action
Copy this template into your current project note and fill in the WDLL line first.
