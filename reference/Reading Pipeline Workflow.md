### Workflow
# Reading Pipeline Workflow

> [!summary]
> Highlights become knowledge only after they become notes you can reuse.

## Step 1: Capture the source
- Save the link/PDF and record why you saved it.
- Add minimal source metadata (author, title, date if relevant).

## Step 2: Create a literature note
Use [[Literature Notes]]:
- Summary (3-7 bullets)
- Key claims (in your words)
- A few short quotes with location if needed

## Step 3: Distill (progressive layers)
Use [[Progressive Summarization]] and [[Progressive Summarization Layer Prompts]].

## Step 4: Promote to permanent notes
Extract 1-3 reusable claims into atomic notes using [[Atomic Note Template]].

## Step 5: Handoff to output
Move the best notes into an outline using [[Outline Handoff Template]].

### Failure mode and fix
Failure mode: You keep collecting sources.

Fix: require a small output (1 permanent note or 1 outline section) before you add more sources.
