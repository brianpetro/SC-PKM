This mini-vault is a guided tour, but the real goal is **3 loops** you can reuse forever:

1) **Capture → Cleanup** (get thoughts out, then structure fast)
2) **Connect → Context** (link at the right granularity; navigate backwards)
3) **Retrieve → Steer** (search/properties to find & drive action)

> Keep one side pane open with **Backlinks** and one with **Outline** while you learn.

## Capstone (5 minutes): "Daily → Project via block embed"
This is the *moment Obsidian stops feeling like a notes app*.

- [ ] Create a note named: **Project - Obsidian 101 Capstone**
- [ ] Add minimal properties at the top:
	- `status: Active`
	- `area: Learning`
- [ ] Open today's daily note ([[08 Daily Notes - A Simple Habit Loop]])
- [ ] Write 3 bullets about what you learned today.
- [ ] Pick the best bullet and add a block id at the end, e.g. `^capstone`
- [ ] In **Project - Obsidian 101 Capstone**, embed that block:

![[YYYY-MM-DD#^capstone]]

- [ ] Open Backlinks on the project note. You should see the trail.

> [!note]
> Some date-to-daily convenience links (via a date property) behave like shortcuts.
> If you want real graph/backlink connectivity, create an explicit `[[YYYY-MM-DD]]` link somewhere relevant.

## Quick vocabulary (as you encounter it)
- **Note:** a markdown file in your vault.
- **Link:** a connection like [[02 Linking - Notes, Headings, Blocks]].
- **Backlink:** an incoming link from another note (watch it update).
- **Properties:** structured metadata at the top of a note.
- **Tags:** lightweight keywords like `#quickstart`.
- **Embed / transclusion:** show content elsewhere without copy-pasting: `![[...]]`
