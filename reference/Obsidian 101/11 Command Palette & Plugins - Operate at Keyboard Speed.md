Your goal: stop hunting menus and start running Obsidian from the keyboard.

## Command palette
- Shortcut: Cmd+P (macOS) / Ctrl+P (Windows/Linux)
- Type to fuzzy-search commands
- Pin the commands you use every day

Try these commands:
- Search: focus
- Templates: insert template
- Daily notes: open today's daily note
- Backlinks: show backlinks
- Graph view: open graph view
- Canvas: create new canvas

## Core plugins
Obsidian ships with core plugins you can enable/disable in Settings.

## Community plugins (optional)
- Turn on community plugins
- Browse, install, then enable

> [!warning]
> Community plugins run third-party code. Install intentionally.
