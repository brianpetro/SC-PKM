> [!summary]
> Track a few habits consistently so you can improve with evidence.

> [!When to use]
> When you want a lightweight way to track habits without apps or complexity.

### Template
```
# Weekly Habit Tracker ({{date:YYYY-[W]WW}})

> [!summary]
> Track 1-3 habits. Review once per week.

## Habits
- Habit 1:
- Habit 2:
- Habit 3:

## Tracker
| Day | Habit 1 | Habit 2 | Habit 3 |
| --- | --- | --- | --- |
| Mon | [ ] | [ ] | [ ] |
| Tue | [ ] | [ ] | [ ] |
| Wed | [ ] | [ ] | [ ] |
| Thu | [ ] | [ ] | [ ] |
| Fri | [ ] | [ ] | [ ] |
| Sat | [ ] | [ ] | [ ] |
| Sun | [ ] | [ ] | [ ] |

## Weekly review (5 minutes)
- Wins:
- Friction:
- One change for next week:
```

### Failure mode and fix
Failure mode: You add 10 habits and stop.

Fix: reduce to 1-3 habits, preferably including one keystone habit (see [[Keystone Habits]]).

### Covers / Does not cover
- Covers: Weekly checkbox tracking for a few habits.
- Does not cover: Complex metrics or streak apps.

## Next action
- Create this week's tracker and mark today.
