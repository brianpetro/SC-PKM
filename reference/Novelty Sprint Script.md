> [!summary]
> A short sprint can restart momentum when starting feels hard.

> [!When to use]
> When you feel stuck, avoidant, or unable to start even small tasks.

### Script
# Novelty Sprint (15 minutes)

> [!summary]
> Use novelty intentionally: a short sprint to break start friction.

## Setup (1 minute)
- Pick one task from your [[3-slot Today Board]].
- Make it smaller: define the first visible step.

## Sprint (12 minutes)
- Set a timer for 12 minutes.
- Do only the first step.
- If you get stuck, write:
	- "Blocked by: ____"
	- "Next attempt: ____"

## Close (2 minutes)
- Write one sentence:
	- "What moved?"
- Choose the next micro-step (or stop).

## Guardrail
Novelty is a tool, not a lifestyle. Use it to start, then return to the main plan.

### Failure mode and fix
Failure mode: You sprint all day and never finish anything.

Fix: use the sprint only to start, then switch to a normal focus block (see [[Deep Block Template]]).

### Covers / Does not cover
- Covers: A short start-friction breaker routine.
- Does not cover: Clinical ADHD guidance.

## Next action
- Run one novelty sprint today on your hardest-to-start task.
