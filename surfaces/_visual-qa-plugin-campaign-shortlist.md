# Plugin campaign shortlist

> [!summary]
> The smallest working set of plugin screenshots that can carry a campaign without forcing reviewers through the full publication-candidate contact sheet.

## Selection rule

- Prefer runtime outcomes over settings, menus, and catalog rows.
- Prefer PKM-relevant note content, retrieval, linking, synthesis, reuse, and action states.
- Include settings/catalog only when they explain a product promise the runtime image cannot show.
- Keep this sheet intentionally small; overflow belongs in [[_visual-qa-plugin-publication-candidates]].
